> App de domótica diseñada para mejorar la autonomía de personas con movilidad reducida. Controla luces, temperatura, humedad y persianas de forma inteligente.

**🌟 Características principales:**
- Automatización por sensores (luces, temperatura, persianas)
- Tres roles: Usuario, Administrador y Familiar
- Chat en tiempo real entre usuarios relacionados
- Monitoreo y gestión de dispositivos en tiempo real

**🛠️ Stack:** Java · Firebase · Angular · Ionic

🔗 [Ver repositorio](https://github.com/Alejandroortega2002/Proyecto_Domotica_ZENSORIUM)

- Publicación de servicios propios
- Filtrado y validación de ofertas para mayor seguridad

**🛠️ Stack:** Java · Firebase · Ionic · Angular

🔗 [Ver repositorio](https://github.com/Alejandroortega2002/AppJobSeeker_TFG_Grado_Superior)
